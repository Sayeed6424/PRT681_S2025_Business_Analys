Tools for BA



* Jira – Track requirements \& user stories (great for Agile)
* Confluence – Team documentation, links with Jira
* Lucidchart / Draw.io – Create workflows \& system diagrams
* Balsamiq / Figma – Build quick wireframes \& UI mockups
* Excel / Google Sheets – Analyze data, track requirements
* SQL tools (e.g., DBeaver) – Query databases, extract insights
* Slack / Microsoft Teams – Real-time team communication
* Trello / Asana – Manage tasks \& project timelines
* Power BI / Tableau – Visualize data with dashboards
* Notion / OneNote – Take notes \& organize personal tasks
