10-Week Plan to Become a Business Analyst



**Week 1: Foundations of Business Analysis**

* Learn the BA role: responsibilities, skills, and deliverables.
* Study SDLC (Software Development Life Cycle) \& Agile.
* Tools: Trello, Jira (for Agile boards).



**Week 2: IT \& System Basics**

* Understand IT systems: web apps, databases, APIs, etc.
* Learn basic terminology used by developers and testers.
* Tools: Draw.io or Lucidchart (for flowcharts, system diagrams)



**Week 3: Requirements Gathering**

* Practice writing Business Requirements Documents (BRD) and User Stories.
* Techniques: Interviews, workshops, surveys, document analysis.
* Agile: Focus on Epics, Stories, Acceptance Criteria.
* Practice scenario: “Build a blog platform” – write requirements.



**Week 4: Introduction to Testing (QA)**

* Learn types of testing: functional, regression, UAT, etc.
* Understand how BAs collaborate with testers.
* Tools: TestRail, Jira (bug tracking)
* Practice writing test cases based on your Week 3 requirements.



**Week 5: Data Analysis \& Visualization**

* Basics of SQL: SELECT, JOIN, WHERE, etc.
* Intro to Excel or Google Sheets: pivot tables, VLOOKUP
* Use Power BI or Tableau for data dashboards.



**Week 6: Process Modeling**

* Learn how to map processes: current state vs future state.
* Use BPMN (Business Process Model and Notation)
* Tools: Lucidchart, Visio
* Practice drawing workflows from simple scenarios.



**Week 7: Use Case \& Wireframing**

* Learn how to create use cases and wireframes.
* Tools: Balsamiq, Figma
* Practice wireframing a user login and dashboard page.



**Week 8: Agile \& Scrum Deep Dive**

* Scrum ceremonies: Sprint Planning, Review, Retro, Standups.
* Understand the roles: Product Owner, Scrum Master, Developer.
* Practice being a BA in a sprint cycle simulation.



**Week 9: Mini Project \& Collaboration**

* Choose a small real-world problem (e.g., a booking system).
* Write requirements, create wireframes, plan test cases.
* Ask a review.



**Week 10: Resume, Portfolio \& Job Preparation**

* Build a portfolio with your mini project.
* Resume: Highlight transferable skills, tools, and project.
* Practice mock interviews (STAR method).
